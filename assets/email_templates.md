# Email Templates

## Initial Outreach

### Template 1: Value-Based
```
Subject: Automate your [process] workflow

Hi [Name],

I help [industry] companies automate their [process].

Quick question: What's the most time-consuming part of your day right now?

Most [industry] businesses I work with struggle with:
- [Pain point 1]
- [Pain point 2]
- [Pain point 3]

If any of that sounds familiar, I can show you how to automate it.

Would you be open to a 15-minute call to see what's possible?

Best,
[Your Name]
```

### Template 2: Question-Based
```
Subject: Quick question about [process]

Hi [Name],

I'm reaching out to [industry] owners in [location] about [process].

Are you currently handling [process] manually, or do you have a system in place?

If you're doing it manually, we can help automate:
- [Benefit 1]
- [Benefit 2]
- [Benefit 3]

Worth a quick conversation?

Best,
[Your Name]
```

## Day 1 Follow-up

```
Subject: Re: [previous subject]

Hi [Name],

Following up on my email from [day/day].

Quick question: Are you currently [action], or do you have a system?

Even a simple automation that:
- [Benefit 1]
- [Benefit 2]

Could save your team [specific time savings] per week.

Would you be open to a [time] call?

Best,
[Your Name]
```

## Day 3 Follow-up

```
Subject: Re: [previous subject]

Hi [Name],

Last follow-up from me.

I've shared some ideas about [topic], but I haven't heard back.

Should I:
- Close your file for now?
- Send over some examples/case studies?
- Try you again in [time frame]?

Let me know what works best.

Best,
[Your Name]
```

```
Looking forward to helping you [solve problem].

```
